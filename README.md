# Bandit-Based Task Assignment (BBTA)

  - [Bandit-based task assignment for heterogeneous crowdsourcing](http://arxiv.org/abs/1507.05800)
  Accepted to [*Neural Computation*](http://www.mitpressjournals.org/loi/neco)

  Submitted.
